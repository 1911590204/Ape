# OptunaTrainer

## Methods

### `__init__`

It shares the same parameters as the [`Trainer`](../trainer.mdx) class, with some additional unique parameters.

**unique parameters:**

- `num_candidates`: Number of candidate few-shot example groups to generate.
- `minibatch_size`: Size of minibatches for evaluation.
- `max_steps`: Maximum number of optimization steps.
