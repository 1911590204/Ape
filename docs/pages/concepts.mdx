# Concepts

Ape is internally composed of two separate libraries.

**`ape-common`** contains common classes necessary for using ape, such as `Prompt`, `Generator`, and `Metric` implementations.

**`ape-core`** contains Prompt Optimization methods implemented as `Trainer` classes.
